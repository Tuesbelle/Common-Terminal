What command is used to list the contents of the current directory?ls

The shell command (ls)is used to list the contents of a directory.

If no arguments are given, it will list the contents of the current working directory.

What command is used to change locations within the file system?

The shell command cdis used to move throughout the filesystem.

It accepts a variey of arguments:
